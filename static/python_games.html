Getting Started

Before you get started with the games, let’s learn together all of the features so that you’ll be able to understand the game instructions:
This part of the page that you will use is called the programming environment.


The left half of the programming environment is where you enter the code. These are the instructions for the program to follow.

Did you notice that some of the lines of code are written in green? That’s because they are comments, which means that those words are “invisible” to the program. These are useful to leave notes for yourself and other programmers reading the code. You can turn a line into a comment by putting a pound symbol # at the start of the line.
Once you’ve written the code, click the run button to see if it works.


If your code is wrong, you will get an error message, that tells you what kind of mistake you made, and you have to fix it and try again.


If your code does not have any mistake that stops the program from running, you will see the result in the visualization window.




Change Myrtle's Color

Moving Myrtle


Here you can tell Myrtle to moves around in the visualization window. Go to line 7 where the code is written. It should have a comment that says “start here”. In the lines below, you will see instructions for Myrtle. In line 8, it says myrtle.forward(50). The first part, before the period, tells the program who you’re talking about. In this case, we’re talking about Myrtle. After the period, you tell the program which way you want Myrtle to face, forward. Then you might notice the (50). Putting that all together, it means line 8 tells Myrtle to face forward and take 50 steps. What about line 10? That mean Myrtle turns to the left 90 degrees, or 1/4 of the way around the circle. Instructions: • Myrtle can walk in two directions: forward and backward • Myrtle can take as many steps as you want her to, but be careful! If you tell her to take more than 200 steps she will walk right off the screen! • Myrtle can turn in two directions: left or right • You can tell Myrtle to turn around as many degrees as you want, but she might get dizzy!





Myrtle Makes a Square


If you haven’t played Moving Myrtle yet, now would be a good time to do that! Because you will need to understand how to move her in order to make the square. It helps to read how Myrtle will move on lines 7 to 13, and imagine yourself walking in a square the way she does. Your assignment is to change the size of the square, by changing how many steps Myrtle goes forward. Notice that since a square has 4 sides, Myrtle will walk forward 4 times, each time she needs to walk the same number of steps. If you don’t do that, she won’t end up making a square! Also, she needs to make a 90 degree turn, so leave all the lines that say “myrtle.left(90)” the way they are, unless you want to be goofy and make a different shape.
